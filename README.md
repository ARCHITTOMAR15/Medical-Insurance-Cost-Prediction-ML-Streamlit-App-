## 📌 Project Overview

This project predicts **medical insurance costs** based on user attributes such as age, BMI, smoking status, and region.

It demonstrates a complete **end-to-end Machine Learning pipeline**, including:
- Data preprocessing
- Feature engineering
- Model training & evaluation
- Overfitting detection
- Model deployment using **Streamlit**
